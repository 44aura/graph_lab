Программа представляет из себя визуализатор графа по данным вершинам и ребрам. На вход передаётся название текстового файла, содержащего информацию о вершинах и ребрах. Результатом является bmp-файл с изображением графа.
#### Как запустить программу?
Сначала, необходимо скачать репозиторий на свой компьютер. Сделать это можно при помощи команды `git clone https://github.com/44aura/graph_lab`
Далее нужно перейти в корневую папку проекта, создать папку build и выполнить команды `cmake ..` и `cmake --build .`
После этого запустите программу командой `./graph example.txt`
